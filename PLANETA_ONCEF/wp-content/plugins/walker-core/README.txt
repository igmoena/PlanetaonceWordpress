== Description ==

Walker Core is the companion plugin for WalkerWP Themes, which provides core functionality and custom post type for the themes. 


== Installation ==

you can install Walker Core Plugins as following two ways:
1. Upload zip file from Dashboard->Plugin->Add New "upload plugin".
2. Extract Walker Core and placed it to the "/wp-content/plugins/" directory.
    - Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==

= Is Walker Core is a free plugin? =

Yes, it is a free plugin.

= Can i use this plugin for any theme?=
This is the companion plugin for themes of WalkerWP so we don't recommended to using with other theme.
